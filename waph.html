Course Information:This course will cover fundamental concepts of web application development, encompassing both front-end technologies like HTML5, JavaScript, and CSS, as well as back-end technologies such as PHP and MySQL. Additionally, it will delve into the realm of web application security, examining vulnerabilities and potential attacks through practical exercises. Strategies for secure programming will also be discussed to mitigate the risk of web application vulnerabilities and attacks.

Lab 1 Overview: This Lab 1 primarily focuses on understanding the practical implementation of how the http request and responses work. Initially, A tool called wireshark is used to capture and analyze the HTTP requests information. Initially Requests are recorded and analysed using the existing simple web server and analysed. Later the same is performed in the terminal using Telnet and then compared with the before results. Later we proceeded on developing the simple CGI program and deployed it to the web server and a html code is also written in C and converted to CGI program and deployed it to the server.Developing web server with PHP is also explored and front end commands like echo, curl are used to provide data through the request and the Request and responses are recorded and analysed with the wireshark tool.
